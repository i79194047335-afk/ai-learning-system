# Coding Rules

## General

- readable > clever
- modular structure
- avoid magic numbers
- comment non-obvious logic
- keep functions focused

---

## File Structure

- one responsibility per file
- descriptive file names
- avoid giant scripts

---

## Dependencies

- minimize unnecessary dependencies
- document all important libraries
- prefer stable tools

---

## AI Collaboration

- AI explains before major refactor
- AI documents generated code
- preserve human readability
