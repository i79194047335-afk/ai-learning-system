# Словарь терминов и понятий

## AI

Artificial Intelligence — искусственный интеллект.

Общее название систем, которые:

* анализируют информацию;
* генерируют текст;
* принимают решения;
* помогают автоматизировать задачи.

---

## LLM

Large Language Model.

Большая языковая модель.

Например:

* ChatGPT;
* Claude;
* Gemini.

Работает с текстом и контекстом.

---

## Agent

Агент.

AI-система, которая:

* выполняет задачи;
* работает с файлами;
* может делать несколько шагов подряд;
* иногда принимает решения самостоятельно.

---

## Multi-Agent System

Мультиагентная система.

Когда несколько AI работают вместе:

* один думает;
* другой пишет код;
* третий ищет информацию.

---

## Workflow

Workflow = рабочий процесс.

Последовательность действий:

```text id="v3tx9n"
идея → AI → файлы → git → GitHub
```

---

## Reasoning

Reasoning = рассуждение.

Способность модели:

* думать;
* анализировать;
* строить логику;
* планировать.

---

## Coding Agent

AI для программирования.

Может:

* писать код;
* редактировать файлы;
* работать с проектом.

---

## Context

Контекст.

Информация, которую AI сейчас «помнит»:

* чат;
* файлы;
* инструкции;
* код.

---

## Context Window

Размер контекста.

Сколько информации модель может держать одновременно.

---

## Long Context

Большой контекст.

Когда модель умеет читать:

* большие документы;
* много файлов;
* длинные диалоги.

---

## Token

Токен.

Мелкая часть текста.

LLM считает текст токенами, а не словами.

---

## API

Интерфейс подключения к AI.

Позволяет:

* программам общаться с моделью;
* подключать AI в IDE и приложения.
